About Us Team Members

This project was made for our Software Engineering class at UC Santa Cruz. The idea was to create a friendly-user interface for League of Legends stats analysis. We want players to benefit from the information we have. Moreover, we want players to use this website often so contact us what we need to improve on. Remember to bookmark us!
